Grid Configuration Info

This test demonstrates the configurable grid overlay functionality:

  • Grid Calculation: The screen is divided into a configurable grid
  • Key Combinations: Each cell gets a unique two-key combination
  • Visual Rendering: Grid cells are rendered with glassmorphism effects
  • Configuration: Rows, columns, opacity, padding, and borders are configurable

Key Combination Pattern:

Keys are generated using home row keys (a,s,d,f,g,h,j,k,l) combined with top row keys (q,w,e,r,t,y,u,i,o,p)

Examples: aq, aw, ae, ar, sq, sw, se, sr, etc.
